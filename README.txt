README:
We upload 3 files for our project:
1.Code.zip, contains all of our project files, including our java code and citystate.csv file. You can run the TestFrame.java in the IDE such as NetBeans. We will also show the process in the demo day.

2.SQL.txt, contains the statement that creates database and tables and contains all the SQL statements that implement all of the functionalities that the project system requires. You can simply copy them together and paste to the SQL command client to run.

3.artificial data.docx, contains the SQL command lines which are used to input many artificial data to our tables. You can simply copy them together and paste to the SQL command client to run.
